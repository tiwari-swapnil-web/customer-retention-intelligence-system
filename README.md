## Customer Retention Intelligence System (CRIS)
Automated system designed to identify, examine, and reduce customer churn

## Problem Statement
Customer churn is a critical issue for subscription-based businesses, leading to revenue loss and increased acquisition costs.

## Objective
To build an end-to-end analytics and machine learning system that identifies at-risk customers and provides actionable retention strategies.

## Approach
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Modeling
- Business Insights & Retention Strategies

## Tech Stack
Python, Pandas, Scikit-learn, SQL (optional), Streamlit (optional)
